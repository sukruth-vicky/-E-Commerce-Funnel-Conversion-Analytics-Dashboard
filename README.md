# E-Commerce Funnel & Customer Behavior Analysis

## Project Overview

This project analyzes customer behavior across the e-commerce conversion funnel using SQL and Power BI.

The analysis focuses on identifying user drop-off points, cart abandonment trends, checkout inefficiencies, conversion performance, and customer retention opportunities.

The project combines product analytics, business intelligence, and operational insights to improve conversion rates and customer experience.

---

## Funnel Stages

- Website Visitors
- Product Views
- Add to Cart
- Checkout
- Purchase

- SQL (MySQL)

- Excel
