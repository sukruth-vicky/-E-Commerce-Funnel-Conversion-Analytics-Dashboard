# Dataset Design Summary

## Dataset Objective

The dataset is designed to analyze customer behavior across the e-commerce conversion funnel and identify user drop-off patterns, conversion inefficiencies, and revenue opportunities.

---

## Planned Dataset Columns

| Column Name | Description |
|---|---|
| User_ID | Unique customer identifier |
| Session_ID | Unique browsing session |
| Device_Type | Mobile/Desktop/Tablet |
| Traffic_Source | Organic/Ads/Social/Referral |
| Product_Category | Product category viewed |
| Product_Viewed | Whether product was viewed |
| Added_To_Cart | Whether item was added to cart |
| Checkout_Started | Whether checkout was initiated |
| Purchase_Completed | Whether purchase was completed |
| Cart_Value | Total cart value |
| Purchase_Value | Final purchase amount |
| Payment_Method | UPI/Card/COD/Wallet |
| Session_Duration_Minutes | User session duration |
| Bounce_Rate_Flag | Whether user exited quickly |
| Repeat_Customer | Yes/No |
| City | Customer city |

---

## Key Analytical Areas

- Funnel Conversion Analysis
- Cart Abandonment Analysis
- Device-wise Conversion
- Traffic Source Performance
- Revenue Analysis
- Repeat Customer Analysis
- Checkout Optimization
